# 代码规范

## 命名

| 文件      | CaseTest   |      |     |

## 函数设计

输入参数在前，输出参数在后。

### 输入参数
1. 输入参数除了基本类型都使用引用。
2. 所有引用都必须是 `const` 的，保证不做修改
### 输出参数
1. 输出参数用指针表示
2. 需要修改的参数用指针表示
